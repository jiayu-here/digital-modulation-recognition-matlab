# 运行说明

## 运行环境

建议环境：

- MATLAB R2024b
- Statistics and Machine Learning Toolbox

本工程的信号调制和 AWGN 加噪由代码直接实现，不依赖 Communications Toolbox 的专用调制函数。

## 一键运行

1. 打开 MATLAB。
2. 将当前文件夹切换到本工程根目录。
3. 在命令行运行：

```matlab
run_modulation_recognition
```

程序会自动完成：

- 构造训练集
- 构造测试集
- 训练 SVM 和 KNN
- 统计准确率
- 生成结果图表
- 保存 `.mat` 模型和数据

## 直接运行源代码

也可以直接打开并运行：

```text
src/run_modulation_recognition.m
```

无论从根目录入口运行还是从 `src` 源代码运行，结果都会保存到工程根目录下的 `results/`。

## 修改常用参数

常用参数在 `src/run_modulation_recognition.m` 文件开头：

```matlab
params.modTypes = ["ASK", "FSK", "BPSK", "QPSK", "16QAM"];
params.snrList = -10:2:20;
params.samplesPerSymbol = 8;
params.symbolsPerFrame = 256;
params.trainFramesPerClassPerSNR = 60;
params.testFramesPerClassPerSNR = 30;
```

如果只想快速演示，可以把：

```matlab
params.snrList = -10:2:20;
```

临时改成：

```matlab
params.snrList = -5:5:20;
```

这样运行更快，但最终报告建议保留 `-10:2:20`。

## 验收标准

运行完成后，MATLAB 命令行应输出不同 SNR 下的准确率表，并且 `results/` 中应生成以下文件：

- `accuracy_by_snr.csv`
- `features_dataset.csv`
- `accuracy_vs_snr.png`
- `time_waveforms.png`
- `spectra.png`
- `constellations.png`
- `confusion_svm_10dB.png`
- `confusion_knn_10dB.png`
- `simulation_models_and_results.mat`
