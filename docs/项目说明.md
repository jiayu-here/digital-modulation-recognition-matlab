# 项目说明

## 项目名称

数字通信系统中的调制识别算法设计与仿真

## 项目目标

本项目用于识别常见数字通信调制方式。系统会生成 ASK、FSK、BPSK、QPSK、16QAM 五类信号，在不同信噪比下加入 AWGN 噪声，提取特征后使用分类器判断调制类型。

## 已实现功能

1. 生成 ASK、FSK、BPSK、QPSK、16QAM 信号。
2. 加入 AWGN 高斯白噪声。
3. 绘制时域波形、频谱图、星座图。
4. 提取幅度均值、幅度方差、相位变化、频率变化、高阶累积量等特征。
5. 使用 RBF-SVM 作为主分类器。
6. 使用 KNN 作为对照分类器。
7. 统计不同 SNR 下的识别准确率。
8. 输出准确率曲线、混淆矩阵、特征数据和模型文件。

## 核心代码

核心代码位于：

```text
src/run_modulation_recognition.m
```

根目录下的：

```text
run_modulation_recognition.m
```

只是运行入口，方便在 MATLAB 工程根目录直接启动。

## 输出结果

运行后结果保存在：

```text
results/
```

主要结果文件包括：

- `accuracy_by_snr.csv`：不同 SNR 下的识别准确率。
- `features_dataset.csv`：训练集和测试集特征。
- `accuracy_vs_snr.png`：准确率曲线。
- `time_waveforms.png`：时域波形。
- `spectra.png`：频谱图。
- `constellations.png`：星座图。
- `confusion_svm_10dB.png`：SVM 混淆矩阵。
- `confusion_knn_10dB.png`：KNN 混淆矩阵。
- `simulation_models_and_results.mat`：模型、参数和数据。
